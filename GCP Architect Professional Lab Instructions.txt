## Google Cloud Professional Architect Lab Hands-On

# Overview
This hands-on exercise is part of the Google Cloud Professional Architect Challenge. The goal is to implement various tasks related to Google Cloud services, configurations, and best practices.

# Setup Instructions
- **Project ID**: Replace `<YOUR_PROJECT_ID>` with your Google Cloud project ID.
- **Zone**: Replace `<YOUR_ZONE>` with the appropriate zone information for your resources.
- **Cluster Name**: Replace `<CLUSTER_NAME>` with your desired cluster name.
- **Replace `<YOUR_INFOARMATION>` with the appropriate information for your resources.

# Setup Instructions

1. **Copy Shell Script File**:
   Download the shell script using curl: 
   curl -LO <My_BASH_SCRIPT_URL>.sh
   Example : curl -LO raw.githubusercontent.com/quiccklabs/Labs_solutions/master/Managing%20a%20GKE%20Multi-tenant%20Cluster%20with%20Namespaces/qwklb.sh

2. **Apply File Permissions**
    sudo chmod +x filename.sh
    Example : sudo chmod +x qwklb.sh
3. **Modify Your Resource Information**
4. **Run the script**
    ./filename.sh
    Example : ./qwklb.sh
